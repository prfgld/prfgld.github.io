I've noticed my node sometimes goes for an extended period of time with reports like "missing delta" or "have delta, but not fully validated". A few times I called "requestblock" to get the missing delta, and sometimes I've restarted the node because that seems to help. Someone else reported to me problems getting a node to initially sync, so that may be related.

Here's another issue I'll pass on from someone else:

The litecoin.conf file might need this line:

rpcbind=127.0.0.1

I don't have this in my litecoin.conf file, so it might depend on the setup.
