How to get some Proofgold bars

by BlakeKeiller, Sunday, July 19, 2020, 13:46 (504 days ago)

Proofgold bars are needed to stake new blocks and to pay transaction fees for sending transactions (including those that publish documents). In the first 730 blocks, it was possible to burn some litecoin to create a Proofgold block, obtaining 25 Proofgold bars in the process. This is no longer possible.

If someone wants a small number of bars to get started, do the following:

Install a proofgold node.

Follow the instructions here to register a nickname corresponding to a public key:

https://proofgold.org/messaging.html

Send a pm to BlakeKeiller from your nickname requesting some bars be sent to an address. For example, in the proofgold console you can do this:

pm YourNick BlakeKeiller "Please send some bars to <youraddresshere>."

You're also free to use "posttop" to post such a request publicly to the forum.

If there end up being too many requests, I'll implement some spam control.

Basic requests should expect to be sent between 0.1 and 1 bars. If your request includes a bug report, you can expect to get between 10 and 100 bars.

Even 0.1 bars is more than enough to publish a document resolving a conjecture with a bounty, so once you have a little proofgold being willing and able to do proofs makes it easy to get more.
