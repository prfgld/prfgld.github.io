Proofgold 0.1.3

by BlakeKeiller, Sunday, June 21, 2020, 10:21 (532 days ago)

The software has been updated to a new version (0.1.3). I included the makevmbytecode script. For those who need to use this, proofgold can be compiled as:

./configure

./makevmbytecode

I have tried this on some systems and sometimes it is more reliable even when the native code (compiled with configure; make) works.

I also included a new configuration/command line parameter "strongrandom." By default it is true, but can be disabled with strongrandom=0 in the proofgold.conf file (or -strongrandom=0 on the command line). If this is combined with the use of -randomseed, then /dev/random will not be used at all.

I have also set aside the first 3000 proofgold bars I mined for bug bounties. If anyone finds a bug in the code (and maybe even provides a fix), then I'll send them 10-100 bars from the bug bounty basket. The amount of bars will depend on my subjective ranking of the bug report on a scale from 1 to 10.
